# Floorplan Line Drawing

A tool for creating and editing floorplan line drawings.

## Features

- Draw and edit floorplan walls
- Create rooms and spaces
- Export/import floorplan designs
- Measure distances and areas

## Getting Started

### Prerequisites

- Python 3.x
- Required Python packages (see requirements.txt)

### Installation

1. Clone the repository
2. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

## Usage

1. Run the application:
   ```
   python main.py
   ```
2. Use the interface to create and edit your floorplan
